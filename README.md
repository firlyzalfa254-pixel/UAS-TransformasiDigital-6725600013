Deskripsi Project

Project ini merupakan implementasi transformasi digital dalam bentuk kuesioner online yang dibuat menggunakan Google Forms. Kuesioner ini bertujuan untuk mengumpulkan data responden secara cepat, efisien, dan terstruktur tanpa menggunakan media kertas.

Melalui Google Forms, proses penyebaran, pengisian, hingga pengolahan data dapat dilakukan secara digital sehingga lebih mudah diakses oleh responden kapan saja dan di mana saja.

Tujuan Project
* Mengimplementasikan konsep Transformasi Digital dalam proses pengumpulan data.
* Mempermudah responden dalam mengisi kuesioner secara online.
* Menghasilkan data yang lebih terstruktur dan mudah dianalisis.
* Mengurangi penggunaan media cetak (paperless).

Fitur
* Formulir online berbasis Google Forms
* Dapat diakses melalui smartphone maupun komputer
* Pengisian data secara real-time
* Penyimpanan data otomatis di Google Spreadsheet
* Mudah dibagikan melalui tautan (link)

Teknologi yang Digunakan

1. Google Forms= Membuat dan mengelola kuesioner
2. Google Drive= Penyimpanan data
3. GitHub= Dokumentasi project
